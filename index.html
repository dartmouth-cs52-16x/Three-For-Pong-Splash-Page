Three for Pong

No pong game? No problem.
With Three for Pong, every night's an on night. Find a game or create a game to meet new friends!

Dartmouth students can create an account using their @dartmouth.edu email address, and specify where on campus they can host games. When a game that has been created is filled, each member of that game will receive a group text message so that they can chat!
